*Extracted from podcast appearances.*

## Insight 1: When you're building a category, you need to make sure that there is a category ...
**Claim:** When you're building a category, you need to make sure that there is a category that's validated by analysts and directory sites and things like that. But also, you want to have a lot of traction in terms of thought leadership like why is this the category? What are the unique value propositions of this particular thing? What are the pain points it solves? And then of course, getting a lot of content around because when you're generating a new category, you're also needing to educate buyers that there is a category that they can now budget for and why they should allocate budget for that.
**Audience:** founder | **Actionability:** 7/10 | **Confidence:** 0.9
**Source:** barbra-gago
**Context:** Barbra explaining the requirements for category creation

**Claim:** Sometimes, competition, you would think like, 'Oh, we want to not have competition, so let's build a new category.'
