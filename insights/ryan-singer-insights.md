## Insight 1: We are not going to start something unless we can see the end from the beginning...
**Claim:** We are not going to start something unless we can see the end from the beginning. We're not going to take a big concept and then say, 'What's the estimate for this thing?' We're going to go the other way around and we're going to say, what is the maximum amount of time we're willing to go before we actually finish something?
**Framework:** Shape Up
**Audience:** operator | **Actionability:** 9/10 | **Confidence:** 0.95
**Source:** ryan-singer
**Context:** Ryan explaining the core principle of shaping: set an appetite (time limit) first, then design a solution that fits within it.
