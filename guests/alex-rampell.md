# Alex Rampell

**Key Beliefs:**
- To determine whether an application is optimized through centralized or decentralized networks, ask: does it benefit from a single source of truth or from distributed trust? (confidence: 0.9)
- Most ICOs today are ridiculous because they issue tokens without a clear need for a decentralized network; tokens should only be used when there is a genuine requirement for a new protocol. (confidence: 0.9)
- When evaluating a startup, look for a clear path to distribution; even the best product fails without a way to reach customers. (confidence: 0.9)
