# Alex Lebrun

**Key Beliefs:**
- Model size matters less than data quality and curation; you can achieve strong performance with smaller models if you have high-quality, domain-specific data. (confidence: 0.9)
- Founders should prioritize building a data moat by creating products that generate unique user interaction data, which can be used to improve models. (confidence: 0.9)
- Startups should focus on vertical AI applications where they can own the data loop and user experience, rather than trying to compete with foundation model providers. (confidence: 0.85)
- For AI products, go-to-market strategy should emphasize solving a specific pain point with a clear ROI, rather than selling the technology itself. (confidence: 0.85)
- Investors should look for AI startups that have a clear path to proprietary data and a strong feedback loop between model improvements and user engagement. (confidence: 0.85)
