# Julie Zhuo

**Role:** Co-founder at Sundial

**Key Beliefs:**
- We need to dissolve the boundaries of these traditional roles and call ourselves builders. I'd love for us to get to the world where that's the title. (confidence: 0.9)
- Management is still really critical. You have a north star, you have a vision, and you're just trying to figure out how to use the resources that you have to get that thing done. Used to be people, but now it's basically models and different models have different strengths. You have to assemble the Avengers so that you can use the right tools for the right purposes. (confidence: 0.9)
- Today management is really about this idea of be sturdy while being flexible. So I think about this metaphor a lot of the willow tree. It can survive a lot of storms, disasters, et cetera, but it's also very flexible. (confidence: 0.9)
- You want to diagnose with data and treat with design. (confidence: 0.9)
- Data is not a tool that's going to tell you what you should build. (confidence: 0.9)
- I don't actually think a lot of the fast growing companies are using data well at this point. Traditionally things just didn't grow that fast. These companies are totally getting by on just good instincts and good vibes, but what always happens is eventually things stop growing. (confidence: 0.9)
