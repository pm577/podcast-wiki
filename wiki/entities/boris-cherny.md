# Boris Cherny

Boris Cherny is the Head of Claude Code at Anthropic, creator of the revolutionary AI coding agent that now authors 4% of all GitHub commits. He previously left for Cursor briefly before returning to Anthropic, drawn back by the mission of AI safety.

## Key Views

### 100% of Code Is AI-Written — and Productivity Is Up 200%
Boris hasn't edited a single line of code by hand since November. He ships 10-30 PRs daily, running five agents simultaneously. Productivity per engineer has increased 200%. He's never enjoyed coding more because he no longer deals with minutiae — the creative, architectural thinking is amplified while the drudgery is eliminated. ^[raw/transcripts/lenny/lenny-2026-02-19-boris-cherny.md]

### The "Builder" Replaces the "Software Engineer"
Boris predicts the title "software engineer" will start to go away, replaced by "builder." By the end of the year, everyone will be a product manager and everyone will code. Coding is becoming a solved problem — the bottleneck shifts to deciding what to build and why. This will be painful for people who define themselves by their coding skill. ^[raw/transcripts/lenny/lenny-2026-02-19-boris-cherny.md]

### Claude Code Is Growing Fastest Not Just Up — But Accelerating
Claude Code's growth rate across any metric continues to accelerate. It's not just going up — it's going up faster and faster. 4% of all GitHub commits (public repositories; private is higher) are now authored by Claude Code, predicted to reach 20% by year-end. The model writing code is just the beginning — next comes autonomous tool use and computer use. ^[raw/transcripts/lenny/lenny-2026-02-19-boris-cherny.md]

### Claude Is Starting to Proactively Generate Ideas
The next frontier: Claude is starting to come up with ideas on its own. It's looking at bug reports and telemetry, forming hypotheses about what needs to be fixed or shipped, and presenting them to the human. This moves beyond code generation toward genuine AI proactivity — behaving more like a coworker than a tool. ^[raw/transcripts/lenny/lenny-2026-02-19-boris-cherny.md]

### Mission (Safety) Over Product Excitement
Boris's most personal insight: he left Anthropic for Cursor because of the exciting product work, but returned within two weeks because he missed the mission-driven culture. "Find someone in the hallway, ask why they're here, the answer is always 'Safety.'" This mission-driven alignment is what makes Anthropic uniquely powerful — and what Boris needs personally to be happy at work. ^[raw/transcripts/lenny/lenny-2026-02-19-boris-cherny.md]
