# Keith Coleman & Jay Baxter

Keith Coleman & Jay Baxter appeared on Lenny's Podcast to discuss keith coleman (vp of product, community notes at x/twitter) and jay baxter (founding ml engineer, community notes at x/twitter), on the bridging-based algorithm, radical transparency, and how community notes reduces misinformation at scale.

## Key Views

**Bridging-Based Algorithm Is the Core Innovation** — A note is only shown when it is rated helpful by people who have historically disagreed with each other. This 'bridging-based agreement' ensures notes are neutral, accurate, and well-written because polarized users cannot find agreement on inaccurate or biased content. The algorithm mathematically requires cross-ideological consensus before any note becomes publicly visible. ^[raw/transcripts/lenny/keith-coleman-jay-baxter.md]

**Radical Transparency and Open Source Build Trust** — The scoring algorithm code is fully open source and all rating data is published daily for anyone to download. External users can replicate the entire service on a single machine to audit it, and researchers like Vitalik Buterin have independently verified the algorithm's behavior. This transparency is not a PR move — it is a structural requirement for a system that mediates public discourse. ^[raw/transcripts/lenny/keith-coleman-jay-baxter.md]

**No Company Override Button By Design** — The system deliberately has no mechanism for the company to change a note's status once it is published. If a problematic note appears, the team must fix the underlying algorithm rather than make exceptions. This constraint reinforces that the product is engineered to work correctly by design, not by central authority. The only way to override a note is to fix the scoring model itself. ^[raw/transcripts/lenny/keith-coleman-jay-baxter.md]

**Open to Everyone Including Polarized Users** — Every post on X is eligible for notes (no exemptions for Elon, advertisers, or world leaders), and contributors include people from extreme viewpoints. This inclusiveness improves the system because it gives the algorithm richer data to identify what notes are truly helpful to a broad audience. The system gets stronger the more diverse the contributor base becomes. ^[raw/transcripts/lenny/keith-coleman-jay-baxter.md]

**Massive Measurable Impact on Misinformation Spread** — A/B tests show a 30-40% drop in engagement (likes/reposts) when a note is shown on a post, and top-line analysis reveals a 50-60% total drop in reposts. Additionally, authors become 80% more likely to delete their noted post, and the median time from post publication to note appearance can be as fast as five hours during major events. The system works at planetary scale with minimal human intervention. ^[raw/transcripts/lenny/keith-coleman-jay-baxter.md]
