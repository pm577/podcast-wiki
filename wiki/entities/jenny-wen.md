# Jenny Wen

Jenny Wen appeared on Lenny's Podcast to discuss why the traditional design process is dead.

## Key Views

**The Design Process is Dead** — 'This design process that designers have been taught — diverge, converge, do research, make beautiful mocks — that's basically dead.' With engineers spinning up 7 Claudes to build features, designers don't have time to make beautiful mocks. The mocking-up part dropped from 60-70% to 30-40% of the role. ^[raw/transcripts/lenny/lenny-2026-03-01-jenny-wen.md]

**Two Types of Design Work** — Stratified into (1) supporting implementation/execution alongside engineering's rapid shipping, and (2) creating 3-6 month vision/direction (not 2-5 year visions). 'You need to point them towards something. If we're all working towards one greater cause, it's much more efficient than random things.' ^[raw/transcripts/lenny/lenny-2026-03-01-jenny-wen.md]

**Ship and Learn Over Perfect Mocks** — With non-deterministic AI models, 'you can't mock up all the states, you can't theorize, you can't even make a clickable prototype. You have to use the actual models and see people try it out — you discover use cases as you see people using them.' Execution velocity beats theoretical design. ^[raw/transcripts/lenny/lenny-2026-03-01-jenny-wen.md]

**Designers Can Code Now** — 'As designers, we now have access to coding tools. I'm implementing all the polish, working with engineers really closely, and prototyping in actual code instead of relying on engineers.' The last-mile implementation is where designers add unique value. ^[raw/transcripts/lenny/lenny-2026-03-01-jenny-wen.md]

**Taste and Judgment Are the Core** — 'At the end of the day, someone has to decide what actually gets built and what matters. Someone still needs to be accountable for the decision.' As AI automates execution, taste and judgment become the irreplaceable human skills. ^[raw/transcripts/lenny/lenny-2026-03-01-jenny-wen.md]
