# Claude Code Guide

## Synthesis

1. **Claude Code is a terminal-based AI coding agent that works differently from chat-based tools.** Unlike Claude chat or ChatGPT, Claude Code operates directly in the terminal, can read and edit files, run commands, create projects from scratch, and manage multi-file codebases. It's designed for building real software, not just generating code snippets.

2. **Non-technical people are using Claude Code successfully.** Lenny's community shared 50+ real-world examples of non-engineers using Claude Code. The common pattern: describe what you want in natural language, Claude Code builds it, and the user tests and iterates. The tool handles the technical implementation — the user provides product judgment.

3. **Getting started requires minimal setup.** Install the Claude Code CLI via npm, authenticate, and you have a terminal-based agent. The key workflow: describe your goal, let Claude Code create files and structure, review the output, and iterate with natural language feedback.

4. **Claude Code excels at full-project creation.** Unlike vibe coding tools like Bolt or Lovable that specialize in web apps, Claude Code can build scripts, automations, CLI tools, data processing pipelines, and full-stack applications. Its terminal-native nature makes it more versatile for technical tasks.

5. **The "vibe coding" workflow applies here too.** The same principles apply: start with a clear description of what you want, review outputs critically, iterate through natural language, and focus on product judgment rather than syntax. The tool handles the "how" — you provide the "what."

6. **Real-world use cases span categories.** From personal productivity tools (time tracking, budget management, note-taking systems) to work automation (report generators, data extractors, Slack bots) to creative projects (interactive art, games, personalized websites) — Claude Code is being used for genuine daily tools, not just experiments.

## Key Disagreements

- **Terminal vs. GUI tools:** Whether terminal-based Claude Code or visual tools like Bolt/Lovable are more accessible for truly non-technical users.
- **Scope of projects:** Whether Claude Code is best for small, well-scoped projects or can handle complex production codebases.
