# Gbrain

Three-layer memory system created by Garry Tan (CEO, Y Combinator). 13K+ GitHub stars. Extends Karpathy's knowledge wiki pattern with search and graph capabilities.

## Architecture

1. **Vector search** (ARR fusion) — semantic retrieval
2. **Backlinks** — wiki-style cross-references
3. **Graph database** — knowledge graph for entity relationships
4. **(Coming) Epistemology system** — track hunches vs beliefs vs world knowledge, attributed to specific people, tracking when hunches manifest into reality

## Philosophy

A founder's journey: have a hunch nobody believes, spend years proving it right. Gbrain should spot that pattern — "this Stanford student believed X and then manifested it." Next step: fully dynamic ontology so different users (researcher, journalist, politician) can have different schemas.
