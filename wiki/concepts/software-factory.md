# Software Factory (AI Era)

The shift from individual coding copilots to an integrated factory that produces production-quality software using AI agents.

## The Transition

- **2025:** Copilots (AI assists individual developers)
- **2026:** Software factory (AI agents plan, code, test, and deploy autonomously with human oversight)

## Key Components

1. **Skills** — reusable runbooks that extract specific capabilities from latent space
2. **Resolvers** — context management for agent instructions
3. **Skillify** — packaging one-off successes into reliable, tested skills
4. **Plan-Eng-Review** — the core workflow loop
5. **80-90% test coverage** — the gate to production

## The Mindset

"I don't open the editor at all anymore." Both Garry Tan and Andrej Karpathy describe building primarily through agent interactions, not direct code editing.
