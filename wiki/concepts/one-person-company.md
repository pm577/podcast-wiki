# One-Person Company (AI Era)

With AI coding agents and agentic systems, a single person can do the work of 500-1,000 people. The "one-person frontier lab" can become a "one-person company."

## Evidence

- Garry Tan: wrote ~1M lines of code with Claude Code in months
- Gstack: 87K GitHub stars, created by one person (not coding 6 months prior)
- YC companies: 6-person teams hitting $10M revenue
- $200/month Claude Code Max plan replaces what used to require $4M and 10 hires

## The Shift

The bottleneck is no longer engineering headcount — it's taste, vision, and the ability to define what to build.
